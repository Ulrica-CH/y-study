# 前端基础

## 概述

前端基础是Web开发的核心技能，包括HTML、CSS和JavaScript三大基础技术。

## 学习路径

### HTML
- 语义化标签
- 表单和交互
- 多媒体元素
- 无障碍访问

### CSS  
- 选择器和优先级
- 盒模型和布局
- Flexbox和Grid
- 响应式设计
- 动画和过渡

### JavaScript
- 基础语法
- DOM操作
- 事件处理
- 异步编程
- ES6+特性
